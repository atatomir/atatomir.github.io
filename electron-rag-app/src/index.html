    Build private RAG pipelines using local models via Ollama or OpenAI.
    Everything can run on your machine. No data leaves your computer.

    2Pull an embedding model (e.g. nomic-embed-text)
    3Pull a chat model (e.g. llama3.2)
    4Create a pipeline, add documents, and ask questions
